# SHNU.CS330
Graphics and Design
Overview

For my final project, I created a 3D interior scene using OpenGL . My goal was to build a scene with multiple shapes, textures, objects and lighting to reflect a selected image. 

Features
Created a complete 3D interior scene
Built objects using primitive shapes
Applied translation, rotation, and scaling to position each object
Added colors and textures to improve realism
Used lighting to make the scene easier to view
Organized the objects into a balanced layout

What I Learned

Building objects was a little challenging, especially with the positioning of the items. I needed to know their starting point and their pivot point. The next most challenging things to learn were lighting and shading to make the scene look more realistic. It was difficult to get the lighting in a correct position without additional shadows or whiting out the scene. Placing the camera & movements, understanding which shapes to use for the scene, and adding texture, while it was a learning process was pretty simple in the end. 

Challenges

The biggest challenge was getting everything positioned correctly. Small changes to translation, scaling, or rotation could completely change how an object looked in the scene. I also spent a lot of time adjusting the placement of objects until the room looked balanced.

Future Improvements

If I continue working on this project, I'd like to add:

Better lighting and shadows
More realistic textures
Simple animations
Interactive camera controls
Higher quality rendering
Reflection

I'm happy with how my final project turned out. It shows how far I have come in the process.  While there's still a lot I'd like to learn; especially when it comes to advanced lighting, shadows, and animation; I feel much more confident working with graphics programming.

I can definitely see myself using these skills again, whether it's for game development, simulations, or just experimenting with personal graphics projects.
